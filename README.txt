Instrucciones para la ejecución de las pruebas de robotFramework

Después de instalar  robot framework y desplegar las pruebas.

1. Abrir una ventana de comando
2. Dirigirse por comandos (cd) a la carpeta donde se encuentran las pruebas desplegadas o descargadas.
3. Escribir la instrucción robot seguido del nombre de la prueba y la extensión .robot
